\# Silins\_UnityUI — Tēla Apģērbšanas Spēle, Andrejs Silins 2026 LTPV



\## Projekta apraksts



2D Unity projekts, kas nodrošina interaktīvu tēla apģērbšanu un aprīkojuma izvēli,

izmantojot Unity UI elementus: Text, Image, Button, Toggle, Slider, Scroll View,

Dropdown, Input Field.



\## Funkcionalitāte



1\. \*\*Sākuma aina\*\* — pogas "Spēlēt" un "Iziet", ainu pārslēgšana

2\. \*\*Vārda/vecuma ievade\*\* — divi Input Field (vārds + gads), vecuma aprēķins

3\. \*\*Tēla izvēle\*\* — Dropdown ar 2 tēliem (Uguns Burvis, Ledus Burvis)

4\. \*\*Apģērbu kategorijas\*\* — 3 kategorijas ar Toggle, katra satur 3 gabalus

5\. \*\*Drag \& Drop\*\* — apģērbu vilkšana uz tēla

6\. \*\*Skaņas efekti\*\* — fona mūzika, pogu klikšķi, tēlu skaņas

7\. \*\*Scroll View\*\* — tēla apraksts, mainās mainoties tēlam

8\. \*\*Slaideri\*\* — garuma un platuma mainīšana
